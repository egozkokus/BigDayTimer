üì• Installation

System Requirements

  • Browser: Google Chrome (version 90 or higher)
  • Operating System: Windows, Mac, Linux, or Chrome OS
  • Internet: Required for installation and premium features

Installation Steps

Step 1: Access Chrome Web Store

Open Google Chrome and navigate to the Chrome Web Store

chrome://extensions

Step 2: Search for BigDayTimer

Use the search bar to find "BigDayTimer" or click our direct link

Step 3: Install Extension

Click "Add to Chrome" and confirm by clicking "Add extension"

üí° Tip: Pin the extension to your toolbar for easy access!

Step 4: Grant Permissions

BigDayTimer requires these permissions:

  • Storage: To save your timer settings locally
  • Active Tab: To display the timer on web pages

Premium Only

Uploading Your Photos

Make your timer truly personal by using your own photos as backgrounds.

Supported Formats

  • JPEG/JPG (recommended)
  • PNG (with transparency support)
  • WebP (modern format, smaller files)
  • Maximum size: 5MB per image

Upload Process

  1. Click "Custom Background" in the timer setup
  2. Select "Upload Image" option
  3. Choose your photo from your device
  4. Preview and adjust positioning if needed
  5. Save to apply to your timer

Image Best Practices

üì∏ Photo Selection

üé® Text Overlay Considerations

üîß Troubleshooting

Common Issues & Solutions

‚ùå Timer Not Appearing

Possible Causes:

  • Extension disabled or removed
  • Timer set to hidden or expired
  • Browser permissions revoked

Solutions:

  1. Check extension is enabled in Chrome settings
  2. Verify timer date/time is in the future
  3. Refresh the webpage
  4. Re-grant extension permissions

üîÑ Timer Showing Wrong Time

Possible Causes:

  • Computer clock is incorrect
  • Timezone mismatch
  • Daylight saving time confusion

Solutions:

  1. Sync your computer's time and date
  2. Check timezone settings in extension
  3. Verify target date/time in timer settings
  4. Account for DST changes in your region

üí∞ Premium Features Not Working

Possible Causes:

  • License verification failed
  • No internet connection
  • Payment processing delay

Solutions:

  1. Check internet connection
  2. Wait 5-10 minutes after purchase
  3. Contact support with your purchase receipt
  4. Try refreshing license in settings

üì± Performance Issues

Symptoms:

  • Slow webpage loading
  • High CPU usage
  • Browser freezing

Solutions:

  1. Reduce number of active timers
  2. Use simpler backgrounds
  3. Disable auto-animations
  4. Restart Chrome browser

Advanced Troubleshooting

üîç Diagnostic Steps

  1. Open Chrome Developer Tools (F12)
  2. Check Console for error messages
  3. Look for BigDayTimer-related errors
  4. Take screenshots of any error messages
  5. Note the exact steps that led to the problem

üìß Getting Help

If issues persist, contact our support team:

  • Email: support@bigdaytimer.com
  • Include: Chrome version, OS, error messages, and steps to reproduce
  • Response time: Usually within 24 hours
